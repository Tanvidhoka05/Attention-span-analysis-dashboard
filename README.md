# 📊 Attention Span Analysis Dashboard (Power BI)

## 📌 Overview

This project analyzes the impact of **short-form content consumption (TikTok, Instagram Reels, YouTube Shorts)** on attention span, stress levels, and productivity across different age groups.

The dashboard provides insights into how **screen time and reel watch behavior** influence cognitive and behavioral patterns.

---

## 🎯 Objectives

* Analyze **attention span trends across age groups**
* Understand **relationship between screen time and stress levels**
* Evaluate **impact of short-form content on productivity**
* Compare usage across **different platforms**

---

## 📂 Dataset

The dataset contains the following features:

* `age`
* `reels_watch_time_hours`
* `daily_screen_time_hours`
* `sleep_hours`
* `attention_span_score`
* `focus_level`
* `task_completion_rate`
* `stress_level`
* `platform`

---

## 🔍 Key Insights

* Higher **screen time correlates with increased stress levels**
* Increased **Instagram reel watch time impacts attention span negatively**
* Attention span is **Improving with age**
* Platform usage influences **behavioral patterns differently**

---
